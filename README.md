Structure du projet
- `Main.java` : point d’entrée du programme, contient la méthode `main`
- `Contact.java` : représente un contact (nom et numéro de téléphone)
- `ContactsManager.java` : gère la liste des contacts (ajout et recherche)
  
Fonctionnement du programme
1.Création d’un objet `ContactsManager`
2. Création de 5 objets `Contact`
3. Attribution d’un nom et d’un numéro de téléphone à chaque contact
4. Ajout des contacts dans le gestionnaire
5. Recherche d’un contact par son nom
6. Affichage du numéro de téléphone du contact recherché


Exemple d’exécution
Si le contact recherché est **Charlie**, le programme affiche :
Numéro de Charlie : 0705024759
